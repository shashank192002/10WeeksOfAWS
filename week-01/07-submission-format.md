# Week 1 Submission Format

Create this folder:

```text
week-01/submissions/YOUR-NAME/
```

Suggested structure:

```text
week-01/submissions/your-name/
├── README.md
├── notes.md
├── code-or-policies/
│   └── custom-s3-readonly-policy.json
├── oidc/
│   └── trust-policy.json
└── screenshots/
    ├── root-mfa.png
    ├── billing-alert.png
    ├── iam-group.png
    ├── iam-user.png
    ├── s3-access.png
    ├── ec2-readonly.png
    └── github-action-success.png
```

## README Template

```markdown
# Week 1 - IAM Challenge

## Name
Your Name

## Topics Practiced
- Root MFA
- Billing alert
- IAM users and groups
- IAM policies
- Least privilege
- Optional GitHub OIDC

## What I Learned
Today I learned how IAM controls access in AWS.

Identity + Permissions = Access.

## Screenshots Added
- Root MFA
- Billing alert
- IAM group
- IAM user
- Access allowed
- Access denied

## Key Takeaway
Least privilege means giving only required permissions, nothing extra.
```
